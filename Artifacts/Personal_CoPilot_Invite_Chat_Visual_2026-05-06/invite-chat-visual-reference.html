Designregler for Invite Chat

  • Chatstrømmen er hovedflaten. Verktøy vises bare som forslag eller via fanen Mer.
  • Assistenten viser ett forslag om gangen og forklarer hva som skjer ved klikk.
  • Tekniske felt som confidence, purposeRef, UUID og signature flyttes til Mer → Avansert.
  • Alle knapper er store nok for touch og må peke til ekte actions før de vises i produkt.
  • Mobil bruker samme tre faner, men med maks tre hurtigchips og ingen sticky composer-avhengighet.
  • Visuelt følger flaten HAVEN-systemet: warm paper, white cards, indigo action, lavender suggestion state.
