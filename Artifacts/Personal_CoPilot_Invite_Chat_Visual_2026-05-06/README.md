# Personal Co-Pilot Invite Chat Visual Reference

Date: 2026-05-06

This artifact converts Claude's textual UX proposal for `Invite Chat` into a
human-visible visual reference. Claude's supplied response describes structure,
copy, tabs, tool routing, privacy, and implementation constraints, but it does
not include a visual design sketch. This mockup is therefore an interpretation
based on:

- Claude's `Invite Chat` UX proposal supplied by Kjetil.
- The canonical HAVEN visual tokens in
  `Deliverables/HAVEN_Conference_CoPilot_Visual_Design_System_2026-05-06.md`.
- Existing Claude visual screenshots archived under
  `Artifacts/Claude_Visual_Design_References_2026-05-06/`.

Files:

- `invite-chat-visual-reference.html` - source HTML/CSS visual reference.
- `invite-chat-desktop-and-mobile.png` - rendered screenshot for humans.
- `invite-chat-mobile-crop.png` - mobile-focused rendered screenshot.
- `render.mjs` - Playwright renderer used to regenerate PNGs.

This is not a runnable product UI. It is a visual acceptance reference for
Porthole/Binding implementation.
